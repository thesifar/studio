# Bhajan Sangam | Spiritual Serenity

A dedicated sanctuary for divine audio and video bhajans, built with a modern and performant tech stack.

## 🚀 Technology Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/) - For optimized performance and server-side rendering.
- **Library**: [React 19](https://react.dev/) - Powering the interactive UI components.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - For a responsive and modern utility-first design.
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/) - High-quality, accessible components built on Radix UI.
- **Generative AI**: [Genkit](https://github.com/firebase/genkit) - Orchestrating AI flows for metadata suggestions and description generation using Google's Gemini models.
- **Icons**: [Lucide React](https://lucide.dev/) - Minimalist and elegant iconography.
- **State Management**: React Context API - Managing the global playlist and play queue.
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Ensuring type safety and robust code quality.

## ✨ Key Features

- **Dual Media Support**: Seamless streaming of both audio and video devotional content.
- **AI-Powered Admin**: Intelligent metadata suggestions and description generation for new content.
- **Dynamic Play Queue**: User-curated playlist management with a persistent glassmorphic interface.
- **Spiritual Aesthetic**: A custom-designed theme using a contemplative purplish-blue and lavender palette.
