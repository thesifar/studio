# **App Name**: Bhajan Sangam

## Core Features:

- Bhajan Browsing: Browse and discover audio and video bhajans categorized by artist, devotional theme, or language.
- Bhajan Playback: Seamlessly play selected audio or stream video bhajans directly within the application.
- Search & Filter: Utilize powerful search and filtering options to quickly find specific bhajans.
- Admin Authentication: Secure login system for administrators to access content management functionalities.
- Admin Dashboard: An intuitive dashboard providing an overview and quick access to all administrative tasks.
- Bhajan Content Management: CRUD (Create, Read, Update, Delete) interface for administrators to manage all audio and video bhajans, including metadata.
- AI Bhajan Description Tool: An AI-powered tool to automatically generate concise and thematic descriptions for newly added bhajans based on title or associated keywords.

## Style Guidelines:

- The chosen light color scheme is inspired by spiritual serenity and traditional Indian aesthetics, evoking a sense of calm and devotion.
- Primary color: A warm, inviting gold (#DBA045), signifying purity, prosperity, and divinity, used for interactive elements and key accents.
- Background color: A soft, creamy beige (#FCF9F3), offering a tranquil canvas for content, promoting readability and focus.
- Accent color: A gentle coral (#ED7E64), used sparingly to draw attention to call-to-actions and provide a harmonious contrast.
- Headline font: 'Alegreya' (humanist serif), providing an elegant and intellectual feel suitable for spiritual and contemplative content.
- Body font: 'Inter' (grotesque-style sans-serif), ensuring excellent readability and a modern, clean aesthetic for longer texts.
- Use subtle and culturally relevant icons, such as a lotus flower for navigation or traditional musical instruments for audio playback, maintaining a clean yet evocative style.
- A clean, responsive, and organized grid-based layout for content display, ensuring easy navigation between different sections and optimal viewing on various devices. The admin panel will feature a clear sidebar navigation.
- Implement subtle and smooth transition animations for page changes and hover effects for interactive elements like play buttons or bhajan cards, enhancing the user experience without distraction.
